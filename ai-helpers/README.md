
# Motivation

This is an attempt to have a single source of truth for guiding LLMs to do my bidding.

It should be usable across machines, agents, and models.

I am still on the fence about skills since they often seem almost redundant. Do I really need to explicitly spell out what TDD is to claude? Shouldnt that be general knowledge. But lets see how this goes.


# Details

For the most part I will copy skills I like out of other community projects that I like and tweak them to be more generic. Then I will symlink the directories so agents can find them.

skills/ - skills for agents

knowledgebase/ - a place to store memories about projects and preferences
